This sql database is exported from MIMP software database. 

To import to your MAMP database:
1, launch MAMP software
2, click "Start Servers"
3, click "optn webstart page"
4, in you browser, this page will automatically opened "http://localhost:8888/MAMP/?language=English"
5, click "php myAdmin" under "Tools" tag
6, click "import", and then choose the sql file
7, success

PS, this is a test version of database, but we can modify it based on this. And you can export the update version of database to a file and push it into github. 

PPS, if you hae a more efficient way to share database, please let us know, thanks!
